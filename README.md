# Jarvis

Static site for the Jarvis project.

## Deployment

This site is configured for Netlify automatic deploys.

- `netlify.toml` publishes the repository root (`.`).
- `index.html` is the site entry point.

## How to push to GitHub

1. Create a GitHub repository manually or using GitHub CLI.
2. Add the remote:

```bash
git remote add origin https://github.com/<your-username>/<repo-name>.git
```

3. Push:

```bash
git push -u origin main
```
